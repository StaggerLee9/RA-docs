# Issue (Issue__c)

## Overview

The Issue object tracks discrete issues or problems that arise within the context of matter management. Issues can represent legal questions, compliance concerns, or operational items that require tracking and resolution.

## Object Details

| Aspect | Description |
|--------|-------------|
| API Name | Issue__c |
| Label | Issue |
| Plural Label | Issues |
| Sharing Model | Private |

## Fields

| Field Name | API Name | Type | Description |
|------------|----------|------|-------------|
| Name | Name | Text | Primary identifier for the issue record |

## Common Use Cases

1. **Issue Tracking**
   - Recording legal or operational issues related to matters
   - Tracking issue resolution status
   - Categorizing issues for reporting

2. **Matter Support**
   - Linking issues to specific matters or investigations
   - Managing issue priorities and assignments

## Best Practices

1. **Issue Management**
   - Use consistent naming conventions for issue records
   - Track issues through to resolution
   - Review open issues regularly

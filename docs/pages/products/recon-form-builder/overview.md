# Product Overview

Recon Form Builder expands the Recon External Intake framework with a configurable form designer. Power users can assemble interactive forms for Experience Cloud sites or internal Lightning pages, publish them without code, and map the inputs directly into the Submission object. Every submission enters a centralized review workspace where internal teams validate details, collaborate on responses, and promote approved records into core Salesforce objects. The result is a governed intake pipeline that keeps sensitive data in Salesforce while delivering responsive forms to the audiences that need them.

## Key Capabilities

- **Drag-and-Drop Form Creation** — Build and update intake experiences quickly so business teams can iterate without a development backlog.
- **Unified Submission Lifecycle** — Route every form response into a shared Submission object that supports triage, approvals, and downstream automation.
- **Flexible Placement** — Embed forms on external community pages for customers or partners, or surface them internally for employee-driven requests.
- **Seamless Data Promotion** — Reviewers can convert approved submissions into Salesforce records, enriching existing processes powered by External Intake.

## Platform Alignment

- Designed as an extension of Recon’s External Intake architecture.
- Honors Salesforce security, ensuring submissions respect sharing and CRUD/FLS controls when promoted.
- Keeps configuration metadata within Salesforce so admins maintain version control and visibility.

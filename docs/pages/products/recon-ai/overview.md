# Product Overview

ReconAI delivers schema-aware conversational intelligence for Salesforce organizations. By combining Agentforce orchestration, Recon Search metadata services, and real-time org context, it translates natural language questions into secure SOQL queries that honor record-level sharing, CRUD, and field-level security. Users can explore data from Salesforce and connected Recon platforms without memorizing filters or building reports—answers arrive in seconds, grounded in the live dataset the user is permitted to see.

## Key Capabilities

- **Natural Language Data Access** — Query Salesforce records or external systems such as Recon DMS using plain language phrases instead of manual filters.  
- **Schema Awareness** — Automatically adapts to each org’s objects, relationships, layouts, and permissions so results align with the organization’s unique data model.  
- **Lightning-Fast Performance** — Cached schema fingerprints (<10 ms) and optimized query pipelines return responses in under two seconds.  
- **Security First** — Every interaction enforces org security (CRUD, FLS, sharing rules) and avoids running raw ad-hoc queries.  
- **Scalable & Extensible** — Built as a second-generation managed package (2GP) with configuration options for admins and pathways for partner integrations.

## Business Impact

- **Time Savings** — Eliminates manual report building and ad-hoc query requests.  
- **Better Decisions** — Surfaces precise, trustworthy answers so teams can act with confidence.  
- **Enhanced Productivity** — Empowers non-technical users to self-serve analytics without expert intervention.  
- **Higher Salesforce ROI** — Extends the value of existing Recon and Salesforce investments by unlocking AI-driven insight.

## Roadmap Insight

- **Visualization Handoff** — ReconAI is exploring automatic JSON payload generation for Recon Data Visualizer so users can pivot from conversational answers to interactive relationship maps.  
- **Release Plan** — Targeting general availability in **Q2 2026**, with early access aligned to the ReconAI pilot program.

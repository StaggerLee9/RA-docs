# Recon Search

Recon Search transforms standard Salesforce page layouts into sophisticated search interfaces. Users can execute comprehensive, metadata-driven searches directly from familiar record pages, seamlessly integrating related lists and external data sources. SWARM automation maintains active search monitoring and delivers proactive notifications when new matching records are identified.

## Package Installation Requirements

- Install the managed package (`Recon Search`, namespace `RSearch`) in your target organization.
- Assign the **Recon Search Admin** permission set to administrators responsible for configuration management.
- Collect integration requirements (API endpoints, authentication credentials, response formats) for each external data provider to be integrated with Recon Search.

# Recon Data Visualizer

Recon Data Visualizer (namespace `ReconDV`) delivers interactive, metadata-driven graphing for Salesforce records. Use the resources below to orient stakeholders, configure the managed package, and keep environments healthy.

## Explore the Documentation

- [Product Overview](./overview)
- [Implementation Checklist](./implementation)
- [Troubleshooting Guide](./troubleshooting)
- [Admin & End-User Guide](../recon-data-visualizer-user-guide)

## Need This Package?

- Contact Recon support to provision the managed package in target orgs.
- Install the latest release listed in your procurement portal (namespace `ReconDV`).
- Assign the Visualizer Admin permission set to system administrators who will maintain configurations.

# ReconAI

ReconAI is an upcoming, schema-aware conversational intelligence layer for Salesforce. Built on Recon Search and Agentforce, it will let users pose natural language questions and receive secure, metadata-driven answers that respect sharing, CRUD, and field-level security.

> **Scheduled Release:** Q2 2026

# Recon Form Builder

Recon Form Builder empowers power users to publish Salesforce-connected intake experiences without custom development. It extends the existing External Intake model so submissions made on Experience Cloud or internal Lightning pages flow into a unified Submission object. Reviewers can triage, collaborate, and promote the vetted data into the broader Salesforce data model—all while keeping form configuration in the hands of business owners.

- Power users who want to create and iterate on forms without relying on code
